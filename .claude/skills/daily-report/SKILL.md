# /daily-report — Отчёт по платежам

## Алгоритм

1. Определи дату (сегодня или указанная)
2. Вызови `list_payments` с created_at_gte и created_at_lte за этот день, limit=100
3. Сгруппируй по статусам
4. Посчитай итого по каждому статусу

## Формат ответа

```
## Отчёт за 2026-03-30

| Статус | Количество | Сумма |
|--------|-----------|-------|
| succeeded | 12 | 45 000 ₽ |
| pending | 3 | 8 500 ₽ |
| canceled | 1 | 2 000 ₽ |

**Итого успешных**: 45 000 ₽ (12 платежей)
```

## Примеры

```
/daily-report
/daily-report 2026-03-29
```
