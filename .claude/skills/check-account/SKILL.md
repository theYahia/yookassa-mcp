# /check-account — Статус магазина

## Алгоритм

1. Вызови `get_balance`
2. Покажи информацию о магазине

## Формат ответа

```
## Магазин ЮKassa

**ID**: 123456
**Статус**: active
**Режим**: тестовый / боевой
**Фискализация**: включена / выключена
```

## Примеры

```
/check-account
```
